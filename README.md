# Aplicativo-de-Calcular-Trelicas
Aplicação para análise de treliças planas que permite ao usuário definir nós, elementos, vínculos e forças, realizando o cálculo estrutural completo. O programa identifica esforços nos elementos (tração ou compressão), forças nos apoios, detecta erros de modelagem (como cruzamento de barras) e destaca elementos com força zero.
